WordSMTP Simple SMTP mailer. Easy & simple setup. All your wordpress / WooCommerce sending emails will be used your verified domain name.

== Description ==

WordSMTP simple SMTP solution to send emails from wordpress to recipient's inbox. All your WordPress emails will use your SMTP host/server to deliver your all emails to recipients.


== Screenshots ==

1. After activated the plugin you should see the WordSMTP Menu image screenshot-1.jpg.
2. Complete the WordSMTP setting options to make it work with your WordPress or WooCommerce in image screenshot-2.jpg.
3. After complete the WordSMTP setting options to be sure test your email setting. If it works fine then you should see success message or if it does not work then your see failed message with all debug info detail in image screenshot-3.jpg.
4. To send your all WooCommerce emails you just need to provide your verified email address or SMTP user address what you provided as SMTP user/login at Setting page. You should go WooCommerce->Settings then under 'Emails' Tab you should see the WooCommerce Email sender options in image screenshot-4.jpg.


== Frequently Asked Questions ==

= What should I do first after activate the plugin? =
First you need to add your SMTP server information to send emails.

= Can I use my own hosting server SMTP? =
Yes you can use your own hosting server SMTP. You can ask your server administrator is there any restrictions / limit to send bulk emails.

= Can I use other mailer server SMTP like Sendinblue, Sendgrid, Amazon SES, Mandrill etc.? =
Yes you can use all of them cause they support SMTP option including you can use any other Mailer server supporting SMTP. In case of other mailer server you should verify your
own domain / sender email address through their settings page.Cause other mailer server do not allow emails until sending email domain is verified / sending email address is verified.
